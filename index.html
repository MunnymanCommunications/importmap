Application Failed to Load

There was a critical error while trying to start the application. This usually happens when a script fails to load correctly.

How to Debug:
1. Open your browser's Developer Tools (usually F12 or Ctrl+Shift+I).
2. Check the Console tab for specific error messages.
3. Check the Network tab to see if any files (like main.tsx) failed to load or have an incorrect "Content-Type" header (it should not be text/plain or text/html).
